CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es el lenguaje de diseño utilizado para definir la presentación, estilo y apariencia visual de documentos HTML o XML en la web. Separa el contenido (HTML) de la estructura y diseño, permitiendo controlar colores, fuentes, márgenes y crear sitios adaptables a diferentes dispositivos.
